爬蟲報告

介紹

爬蟲是一種程式，用於自動訪問網頁並從中收集信息。本報告探討了一個簡單爬蟲的設計和實現。

設計

本爬蟲程式設計用於收集指定網站的頁面信息。以下是爬蟲程式的主要設計特點：

  • 目標網站： 選擇了一個目標網站進行爬取。
  • 程式語言： 使用 Python 語言進行開發。
  • 庫的選擇： 使用了 Requests 库進行網頁的 HTTP 請求，以及 BeautifulSoup 库來解析和提取 HTML 元素。
  • 主要功能： 爬蟲程式能夠訪問網站的不同頁面，提取指定的資訊如標題、內容等，並將這些資訊存儲或顯示。

程式碼示例

以下是爬蟲程式的部分 Python 代碼示例：


import requests
from bs4 import BeautifulSoup

# 定義爬取函式
def crawl_website(url):
    # 發送 HTTP GET 請求
    response = requests.get(url)
    
    # 解析 HTML 內容
    soup = BeautifulSoup(response.content, 'html.parser')
    
    # 提取標題
    title = soup.find('title').text
    
    # 提取內容
    content = soup.find('div', class_='content').text
    
    return title, content

# 主程序入口
if __name__ == '__main__':
    target_url = 'http://example.com'
    title, content = crawl_website(target_url)
    
    print(f'Title: {title}')
    print(f'Content: {content}')
